This an example of the API that is generated for a component in .60 design markup. This may help you understand what functions you can call and how you can pass data in and out. This is the source code:

SampleComponent := Window {
    property<int> counter;
    property<string> user_name;
    callback hello;
    /// ... maybe more elements here
}
