import Link from '@slint/common-files/src/components/Link.astro';

A `VerticalBox` is a [VerticalLayout](/1.17.0/docs/slint/reference/layouts/verticallayout.md) with default `padding` and `spacing` from [StyleMetrics](/1.17.0/docs/slint/reference/std-widgets/globals/stylemetrics.md),
providing theme-appropriate values. Use `VerticalLayout` directly when you need full control over spacing and padding.
