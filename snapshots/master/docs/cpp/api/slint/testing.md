Use the functions and classes in this namespace for in-process UI testing.

This module is still experimental - its API is subject to changes and not stabilized yet. To use the module, you must enable the `SLINT_FEATURE_EXPERIMENTAL=ON` and `SLINT_FEATURE_TESTING` CMake options.
