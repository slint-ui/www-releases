`PointerEventKind`

 The enum reports what happened to the `PointerEventButton` in the event

* **`cancel`**:  The action was cancelled.
* **`down`**:  The button was pressed.
* **`up`**:  The button was released.
* **`move`**:  The pointer has moved,
