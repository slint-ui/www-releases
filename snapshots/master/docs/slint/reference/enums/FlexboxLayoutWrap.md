`FlexboxLayoutWrap`

 Controls whether flex items wrap onto multiple lines.

* **`wrap`**:  Flex items wrap onto multiple lines, from top to bottom (for row direction) or left to right (for column direction).
* **`no-wrap`**:  All flex items are laid out on a single line (default for CSS, but Slint defaults to `wrap`).
* **`wrap-reverse`**:  Flex items wrap onto multiple lines in the reverse direction.
