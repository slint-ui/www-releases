---
import Link from '@slint/common-files/src/components/Link.astro';

A `GridBox` is a [GridLayout](/master/docs/slint/reference/generated/layouts/grid-layout.md) with default `padding` and `spacing` from [StyleMetrics](/master/docs/slint/reference/std-widgets/globals/stylemetrics.md),
providing theme-appropriate values. Use `GridLayout` directly when you need full control over spacing and padding.
