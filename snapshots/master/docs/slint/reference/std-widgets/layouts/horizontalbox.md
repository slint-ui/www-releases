import Link from '@slint/common-files/src/components/Link.astro';

A `HorizontalBox` is a [HorizontalLayout](/master/docs/slint/reference/generated/layouts/horizontal-layout.md) with default `padding` and `spacing` from [StyleMetrics](/master/docs/slint/reference/std-widgets/globals/stylemetrics.md),
providing theme-appropriate values. Use `HorizontalLayout` directly when you need full control over spacing and padding.
