Expand description

Register a font from byte data dynamically.

The argument is the owner of a slice of byte that contains the data of the font. It can for example be a &'static [u8] obtained from include_bytes!() or a Vec<u8> obtained from std::fs::read().

The data must be in the TrueType (TTF) format.

let font_bytes = include_bytes!("sharedfontique/DejaVuSans.ttf");
slint::register_font_from_memory(font_bytes).expect("Failed to register font");
